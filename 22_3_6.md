## git
1. git 을 초기화하고 md  파일을 만든다. 작성한다(git init)
2. 그러고 나서 깃허브에 repository를 새로 생성하고 주소를 복사.
3. 제일 처음만 하면 되는데, 제어판->자격증명에 가서 git.com의 이름과 비번을 적어 놓는다.
4. 보통 이름은 깃의 메인화면 내 아이디 이고
5. 비번은 요즘은 tocken을 발급받는데 깃의 세팅=> developer setting->으로 가서 tocken을 발급받아
6. 3번의 자격증명 비번에 붙여 넣는다. 그럼 자동으로 연결 된다.
- 주의 : repository의 주소만 복사 해서 붙여 넣어야지 폴더명까지 붙여 넣으면 안된다. 보통 폴더 명은
걍 에디터에서 작성한다.
ex)  '''git remote add til https://github.com/Jeongyungeun/TIL.git'''
